
## 使用npm
### 检查npm是否安装
```bash
npm -v
```
若出现版本号,则可以使用npm相关命令。

### 安装命令[npm run clear](https://github.com/dgiot/dgiot_dashboard/blob/master/package.json#L19)

```
# 项目根目录
npm run clear
```

### 运行命令

```
# 项目根目录
npm run dev
```

### 打包命令

```
# 项目根目录
npm run build
```

## 使用cnpm
### 检查cnpm是否安装
```bash
cnpm -v
```
若出现版本号,则可以使用cnpm相关命令。
