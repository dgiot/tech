## 菜单管理
菜单包括各个页面的文件路径,用户可以自定义路径实现自己的功能模块.
用户通过点击右侧按钮实现子菜单的自定义操作.也可用于页面路径的查找.
利用上排导航栏按钮对菜单进行快速操作.
